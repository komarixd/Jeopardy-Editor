# Jeopardy Game Editor
![Status](https://img.shields.io/badge/Status-Stable-green)
![Ststus](https://img.shields.io/badge/.NET-4.7.2-blue)

A WPF desktop application designed to create, edit, and manage question sets for the Jeopardy Game engine.

## ✨ Features
* **Dynamic Board Geometry:** Unlike standard Jeopardy, this editor supports custom maps with any number of rows and columns.
* **Interactive Canvas Editor:** 
    * **Drag-and-Drop:** Move elements freely across the canvas.
    * **Live Resizing:** Scale UI elements dynamically.
* **Media Support:** Attach images or sound clips to specific clues.

## 🚀 Getting Started
1. Download the latest version from the [Releases](../../releases) page.
2. Extract the ZIP file.
3. Run `jeopardy.exe`.
